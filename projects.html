  • Home IoT using the NodeMCU (ESP8266) - (Feb - March 2018)

    Industry: Qantom Software Pvt. Ltd.
    1. Connect to a registered and existing Access Point (AP).
    2. On successful connection, webserver tasks would listen on port 80 and accordingly, the GPIO pins would be controlled (setting the NodeMCU itself as an access point).
    3. This can be extended to smart home applications and in hospitals, making it easier for patients with (limited) motor control to command their immediate surrounding via smart devices. Learn More

  • Automated Smart Switch using relays and Arduino Uno - (Sept - Dec 2017)

    Industry: Voltronex Technology Pvt. Ltd.
    1. Successful Design and materialisation of a compact device for switching between manual control (from the switch-board) or the sensor control (by the micro-controller) for home appliances.
    2. The user is thus given the real-time option to allow sensors to control the home appliances or else manually override the micro-controller and switch the appliances from the conventional switch-board.
    3. Can be implemented in hospitals (given the wireless communication) or smart homes. Learn More

  • Temperature or Light Controlled Switch using the LM35 and LDR

    Designed and implemented an analog integrated circuit to automate switching of an LED based on LDR and also 12V fan based on the temperature sensor. When the ambient light intensity is negligible, then the LDR triggers the LED to switch ON. Likewise, the temp. sensor (LM35) triggers the fan when the temperature rises above a preset threshold as set by the potentiometer. Learn More

  • Water Level Controller using the Arduino Uno (atmega328P) and UltraSound Sensor (40MHz)

    To address overflow of water from household overhead tanks, a prototype to automatically switch the motor pump ON/ OFF was built. The Arduino (Atmega 328P,   8 bit) and an Ultrasonic sensor (40 MHz) were used to monitor the water level of the overhead tank. Learn More
